# PlantBased.Blog

A Jekyll-powered blog dedicated to plant-based living, vegan recipes, and sustainable lifestyle choices.

## About

PlantBased.Blog is a resource for anyone interested in plant-based living, whether you're a long-time vegan, just starting out, or simply curious about incorporating more plants into your diet.

## Development

This blog is built with [Jekyll](https://jekyllrb.com/), a static site generator.

### Prerequisites

- Ruby (3.x or higher)
- Bundler

### Setup

1. Install dependencies:
   ```bash
   bundle install
   ```

2. Build the site:
   ```bash
   bundle exec jekyll build
   ```

3. Serve the site locally:
   ```bash
   bundle exec jekyll serve
   ```

4. Visit `http://localhost:4000` in your browser

### Adding Content

- Blog posts go in the `_posts` directory
- Use the format `YYYY-MM-DD-title.markdown` for post filenames
- Edit `_config.yml` for site-wide settings
